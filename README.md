# LogicFun 演示页面

> 🌐 **在线访问**: [https://logicfun.github.io](https://logicfun.github.io)

这是 LogicFun（逻辑乐园）应用的官方演示页面，展示了应用的所有功能和游戏。

## ✨ 特性

- 🎨 现代化渐变色设计
- 📱 完全响应式布局，支持移动端
- 🖼️ 横向滚动 Gallery 展示应用截图
- 🔍 Lightbox 图片查看器
- 🎭 流畅的动画和交互效果
- 🎮 8款精选逻辑益智游戏展示

## 📁 项目结构

```
logicfun.github.io/
├── assets/          # 资源文件（CSS、JS）
├── images/          # 应用截图和图标
├── docs/            # 文档
├── index.html       # 主页面
├── privacy.html     # 隐私政策
├── terms.html       # 用户协议
└── favicon.png      # 网站图标
```

## 🚀 快速开始

详细的部署指南请查看：[docs/DEPLOYMENT.md](docs/DEPLOYMENT.md)

**快速部署步骤**：
1. 创建 GitHub 仓库 `logicfun.github.io`
2. 上传所有文件到仓库
3. 在 Settings > Pages 中启用 GitHub Pages
4. 访问 `https://logicfun.github.io`
